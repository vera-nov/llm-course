# Задание

Реализовать RAG-систему на основе датасета [rus_xquadqa](https://huggingface.co/datasets/bearberry/rus_xquadqa) и оценить работу LLM с ней и без нее.

## Этапы выполнения

1. Собрать все уникальные текстовые сегменты `chunk` из поля `context` в один корпус.
2. Построить по этому корпусу векторный индекс в `Qdrant :memory:`.
3. Через LangChain сделать функцию поиска.
4. Реализовать две версии QA с RAG и без.
5. Оценка на вопросах из датасета: получить ответ модели с RAG и без.
6. Сравнение качества - посчитать лексические метрики.
